**Project Overview
**
The Task Management Web Application is a simple and interactive web-based project developed using HTML, Bootstrap 5, and JavaScript.
This application allows users to add, edit, and manage tasks along with time details using a clean interface and modal pop-ups.

This project is created as part of a college mini project to demonstrate practical knowledge of DOM manipulation, form handling, and Bootstrap modals.
